## 如何更改語言

### 在 Chloros 桌面/瀏覽器中

1. 打開應用程序設置
2. 導航至語言選擇菜單
3. 從列表中選擇您的首選語言
4. 界面會立即更新

### 在 Chloros CLI 中

在命令中使用語言參數來設置界面語言。

### 在 Chloros Python SDK 中

初始化 SDK 時設置語言參數，以您的首選語言獲取消息和輸出。

## 覆蓋範圍

全面支持所有 38 種語言：

* **Chloros Desktop** - 完整的 GUI 翻譯
* **Chloros 瀏覽器** - 所有語言的 Web 界面
* **Chloros CLI** - 命令行界面和輸出消息
* **Chloros Python SDK** - API 消息和文檔

語言支持確保世界各地的用戶可以無障礙地使用其母語高效工作。
