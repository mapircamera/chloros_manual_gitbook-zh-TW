## 如何更改语言

### 在 Chloros 桌面/浏览器版

1. 打开应用程序设置
2. 进入语言选择菜单
3. 从列表中选择所需语言
4. 界面将即时更新

### 在 Chloros CLI 中

通过命令中的语言参数设置界面语言。

### 在 Chloros Python SDK 中

初始化 SDK 时设置语言参数，即可获取目标语言的消息与输出。

## 覆盖范围

全部 38 种语言在以下场景均获完整支持：

* **Chloros Desktop** - 完整图形界面翻译
* **Chloros Browser** - 全语言网页界面
* **Chloros CLI** - 命令行界面及输出消息
* **Chloros Python SDK** - API 消息与文档

语言支持确保全球用户能够无障碍地使用母语高效工作。
