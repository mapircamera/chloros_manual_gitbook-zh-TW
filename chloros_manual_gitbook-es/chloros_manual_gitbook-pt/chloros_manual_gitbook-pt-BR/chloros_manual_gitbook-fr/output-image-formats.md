# 輸出圖像格式

### TIFF（16 位）

16 位（像素數字編號 0 - 65535）TIFF (.tif) 格式輸出。

### TIFF（32 位，百分比 %）

32 位（像素浮點數 0.0 - 1.0）TIFF (.tif) 格式輸出。像素值表示反射率百分比，其中 0.0 = 0%，1.0 = 100%。

### **JPG（8 位）**

8位（像素數字位數0 -255）JPG（.jpg）格式輸出。

### **PNG（8 位）**

8 位（像素數字 0 - 255）PNG (.png) 格式輸出。無損壓縮適合網頁查看和可視化。
