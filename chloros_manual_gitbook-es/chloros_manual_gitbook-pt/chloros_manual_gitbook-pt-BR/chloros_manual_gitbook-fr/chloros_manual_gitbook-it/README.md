# Chloros 用戶手冊 - 意大利語

這是 Chloros Professional 多光譜圖像處理軟件用戶手冊的意大利語翻譯。

## 關於克洛羅斯

Chloros 是專業的多光譜圖像處理軟件，可通過強大的桌面軟件、命令行工具和 Python API 將圖像轉換為可操作的見解。

### 特徵
- 桌面和瀏覽器界面
- CLI 和 Python SDK (Chloros+)
- 有 28 種語言版本
- 24+ 植被指數
- 反射率校準
- 環境光傳感器支持

## 文件

本手冊與GitBook同步。該文檔涵蓋：
- 安裝和設置
- 圖像處理工作流程
- 校準程序
- 植被指數計算
- CLI 和 Python SDK 使用
- API參考
