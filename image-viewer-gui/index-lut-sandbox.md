# 指数/LUT 沙盒

指数/LUT 沙盒是 Chloros 图像查看器中的一个交互式工作区，可让您实时尝试多光谱指数计算和色彩可视化。这款强大的工具能帮助您测试不同的指数、优化数值范围，并创建可直接用于发表的可视化成果，而无需重新处理整个数据集。

## 什么是指数/LUT 沙盒？

### 用途

沙盒提供：

* **实时指数计算** - 即时应用任何植被指数
* **交互式 LUT 调整** - 微调颜色渐变和范围
* **工作流优化** - 在批量处理前确定最佳设置

### 沙盒与项目处理的对比

**指数/LUT 沙盒（交互式）：**

* 每次处理单张图像
* 即时反馈
* 适用于实验和迭代
* 不对文件进行永久性更改
* 非常适合探索和测试

**项目处理（批处理）：**

* 一次性处理整个数据集
* 预配置设置
* 生成永久性输出文件
* 耗时较长
* 适用于设置已最终确定的情况

{% hint style="success" %}
**最佳工作流程**：使用沙盒进行实验并找出最佳的指数和LUT设置，然后在项目处理中将这些设置应用于整个数据集。
{% endhint %}

## 使用索引/LUT 沙盒

### 了解预计算索引

在 Chloros 中，可在项目处理期间应用索引。要确定希望对导出数据应用哪些索引和 LUT 设置，最简便的方法是使用图像查看器沙盒。

沙盒允许您：

* **应用新的索引和颜色渐变（LUT）**以可视化数据
* **交互式调整可视化设置*** **查看**已计算的索引图像
* **检查**所有缩放级别下的像素值

### 打开沙盒

可通过**图像查看器** <img src="../.gitbook/assets/icon_image-viewer.JPG" alt="" data-size="line"> 侧边栏标签页中：

1. 在文件浏览器图像网格中点击一张图像，它将在 **图像查看器** <img src="../.gitbook/assets/icon_image-viewer.JPG" alt="" data-size="line"> 标签页中打开
2. 点击 **图像查看器** <img src="../.gitbook/assets/icon_image-viewer.JPG" alt="" data-size="line"> 选项卡，若尚未打开，将弹出左侧侧边栏

### 选择要应用索引/LUT 的图像

要在图像查看器 <img src="../.gitbook/assets/icon_image-viewer.JPG" alt="" data-size="line"> 沙盒中操作索引：

1. 点击主图像网格中的**图像**将其打开
2. 随后将打开**图像查看器** <img src="../.gitbook/assets/icon_image-viewer.JPG" alt="" data-size="line"> 选项卡随即打开
3. 点击**图层下拉菜单**（查看器右上角）
4. 从下拉菜单中选择图层：
   * RAW（反射率）

### 向图像应用索引

当图像处于全屏模式且**图像查看器** <img src="../.gitbook/assets/icon_image-viewer.JPG" alt="" data-size="line"> 标签页侧边栏打开后：

1. 勾选侧边栏顶部的“索引”复选框
2. 从左侧下拉菜单中选择您的相机滤镜
3. 从右侧下拉菜单中选择所需的索引公式
4. 将滤镜通道的彩色圆圈拖动到下方索引公式中的相应位置
5. 公式有效后，图像将更新并显示索引值
6. 移动鼠标光标，查看光标所在位置的数值
7. 放大以查看单个像素及其关联值

每个指数都有特定的数值范围和含义：

#### NDVI 示例

```

Formula: (NIR - Red) / (NIR + Red)

For Survey3W RGN camera:
NIR = 850nm band
Red = 661nm band

Result range: -1.0 to +1.0
Typical vegetation: 0.4 to 0.9
Stressed vegetation: 0.2 to 0.4
Bare soil: 0.0 to 0.2
Water: -0.1 to 0.1
```

有关完整的指数公式文档，请参阅 [多光谱指数公式](../project-settings/multispectral-index-formulas.md)。

## 使用 LUT（查找表）

### 什么是 LUT？

**查找表（LUT）** 将数值索引映射为颜色以便可视化：

* **输入**：索引像素值（例如，NDVI 0.65）
* **输出**：RGB 颜色（例如，亮绿色）
* **目的**：使模式更易于观察和解读**灰度 LUT 与彩色 LUT 的区别：**

* 灰度：科学且中性，显示原始数据
* 彩色 LUT：直观且具有视觉冲击力，突出显示模式和差异

{% hint style="success" %}
**可视化优势**：对灰度索引图像应用彩色 LUT，可显著提升识别模式、异常及关注区域的便捷性，一目了然。
{% endhint %}

### 对索引图像应用 LUT

当您拥有显示以下内容的索引图像后

1. 点击 <img src="../.gitbook/assets/image (1) (1).png" alt="" data-size="line"> “+添加 LUT”按钮
2. 选择颜色渐变
3. 调整剪切的最小/最大端点
4. 调整剪切模式
5. 在 **图像查看器** <img src="../.gitbook/assets/icon_image-viewer.JPG" alt="" data-size="line"> 标签页侧边栏中的“索引”复选框以应用 LUT

### 选择颜色渐变

**选择渐变：**

1. 在 LUT 面板中，找到**彩色渐变条**

2. 将鼠标悬停在其上以查看可用的渐变预设
3. 选择所需的渐变
4. 当勾选“索引”复选框时，图像将**立即更新**为新颜色

{% hint style="success" %}
**最佳实践**：对于 NDVI 等植被指数，Red-黄色-Green 渐变最为直观，因为它符合自然的色彩关联（绿色=健康，黄色=中等，红色=受胁）。
{% endhint %}

### 调整颜色类别

**“类别”控件**决定渐变中显示的离散颜色阶数：**类别数量选项：*** **2-5 个类别**：非常宽泛的分类，区域分明
* **6-10 个类别**：平衡，适合分类
* **11-20 个类别**： 平滑的渐变，外观连续
* **20+ 个类别**：近乎连续，平滑度最高**调整方法：**

1. 在 LUT 面板中，找到**渐变条下方的颜色色块**

2. 点击 + 按钮增加类别数量
3. 双击颜色色块减少类别数量
4. 图像上的渐变会**实时**更新**对可视化的影响：*** **较少类别**（3-5）：形成鲜明分区，分类简化，更易区分类别
* **中等类别**（6-10）：平衡方案，适用于大多数应用
* **更多类别**（15-20）：平滑过渡，细节丰富，呈现照片般的外观**适用场景：*** **少分类（3-5）**：演示文稿、分类地图、简易报告
* **中等分类（6-10）**：常规分析、细节平衡、标准报告
* **多分类（15-20）**：科学分析、详细检查、出版级输出

### 调整数值范围

**值范围控件**决定渐变中哪些索引值映射到哪些颜色：**LUT 面板中的范围控件：*** **最小值**：颜色尺度的下限
* **最大值**：颜色尺度的上限
* **中间值**：自动分布在最小值和最大值之间（基于类别数量）

#### 调整最小/最大值

**调整数值范围的方法：**

1. 在 LUT 面板中，找到**最小值**和**最大值** 输入框
2. 点击 **最小值** 输入框
3. 输入所需的最小值（例如：`0.2`）
4. 按下 **Enter** 或点击字段外部
5. 重复上述步骤设置 **最大值** 字段（例如：`0.9`）
6. 可视化效果将**立即更新**{% hint style="info" %}**自动缩放**：首次应用 LUT 时，Chloros 会自动将最小值/最大值设置为图像中的实际数据范围。随后您可以缩小该范围，以聚焦于感兴趣的特定数值范围。
{% endhint %}

**NDVI 范围调整示例：*** **全范围**：`-1.0` 至 `1.0` （显示所有可能的值）
* **聚焦植被**：`0.2` 至 `0.9` （排除裸露土壤和水体）
* **仅健康植被**：`0.5` 至 `0.9`（仅突出显示生长旺盛的植物）
* **压力检测**：`0.2` 至 `0.5` （突出显示问题区域）
* **自定义范围**：根据您观察到的像素值进行调整**为何要调整范围？*** **提高**目标区域的对比度
* **排除无关值**（例如水体、裸露土壤）
* **标准化**多张图像或不同日期间的可视化效果
* **突出显示**窄值范围内的细微差异

### 裁剪超出范围的值

当像素值超出您定义的最小/最大范围时，您可以通过**裁剪模式**控制其显示方式。

#### **可用的裁剪模式选项：**

#### 1. 最小值和最大值

* **低于最小值**的像素 → 使用渐变中的**第一种颜色**显示（例如红色）
* **高于最大值**的像素 → 使用渐变中的**最后一种颜色**显示（例如绿色）
* **应用场景**：突出显示极端值，在边界处使用饱和色展示完整数据范围
* **示例**：NDVI 值低于 0.2 的全部显示为红色，高于 0.9 的全部显示为绿色

#### 2. 透明背景

* **超出范围**的像素将变为**完全透明*** 仅**在范围内的**像素显示颜色渐变
* **应用场景**：GIS 叠加层、隔离特定数值范围、仅突出显示感兴趣区域
* **示例**：仅显示 NDVI 0.4-0.7 范围内的颜色，其余部分透明

{% hint style="warning" %}
**透明度限制**：透明像素在查看器中将显示为背景色。处理过程中导出时，PNG 格式会保留透明度，但 JPG 格式不会。
{% endhint %}

#### 3. 索引背景

* **超出范围**的像素以**灰度**显示（显示原始索引值）
* **在范围之内**的像素显示**颜色渐变*** **应用场景**：细微高亮，在强调感兴趣区域的同时保持上下文
* **示例**：用颜色突出显示受胁迫的植被（NDVI 0.3-0.5），同时将健康区域显示为灰色

#### 4. 原始背景

* **超出范围**的像素显示**原始多光谱图像*** **在范围之内**的像素显示**颜色渐变*** **应用场景**：最直观——将自然图像背景与分析性色彩叠加层相结合
* **示例**：在实际田间/作物外观上叠加彩色编码的受胁区域

### 选择合适的剪切模式

| 剪切模式              | 最适合                                   | 可视化风格          |
| -------------------------- | ------------------------------------------ | ---------------------------- |
| **最小值和最大值**    | 全数据显示，科学分析     | 所有像素着色           |
| **透明背景**         | GIS 叠加，隔离特定范围    | 范围内的像素着色，超出范围为空白 |
| **索引背景**       | 微妙强调，保持数据上下文  | 范围内着色，范围外灰度  |
| **原始背景**    | 报告、演示、直观分析 | 范围内着色，范围外保留原图 |

### 创建自定义 LUT 颜色

若需完全掌控可视化效果，您可以通过编辑单个颜色停靠点来创建 **自定义颜色渐变**。**创建自定义渐变的方法：**

1. 在 LUT 面板中，找到**渐变预览条**

2. 查看渐变下方**的色块方块**

3.**点击某个色阶** 进行选择
4. 将打开一个 **颜色选择器**

5. 使用以下方式选择新颜色：
   * **色轮**：可视化颜色选择
   * **RGB/HSV 滑块**：精确颜色控制
   * **十六进制代码输入**：精确颜色指定（例如，`#FF0000` 表示红色）
6. 点击颜色选择器外的区域 **以应用新颜色**

7. 图像上的渐变色**会立即更新**

**添加或删除颜色停靠点：*** **添加停靠点**：点击 + 图标在末尾添加新色块
* **删除停靠点**：双击颜色方块以删除该色块**自定义策略：*** **反转渐变**：颠倒颜色顺序以改变含义（例如，绿色=低，红色=高）
* **品牌配色**：使报告颜色与贵组织的配色方案相匹配
* **色盲友好**：使用橙蓝或紫黄配色方案
* **打印优化**：选择在彩色和灰度打印中均适用的颜色
* **多阈值**：在特定数值阈值处使用不同颜色进行分类

{% hint style="info" %}
**保存自定义渐变**：自定义渐变可保存并重复使用。点击 LUT 面板中的保存图标，即可保存自定义配色方案以备将来使用。
{% endhint %}

## 交互式工作流程

### 实时更新

沙盒中的所有 LUT 调整都会**即时且交互式地**更新图像：

* **切换图层** → 图像立即变化
* **选择渐变** → 颜色即时更新
* **调整数值范围** → 对比度实时变化
* **更改类别** → 渐变平滑度立即更新
* **修改剪切** → 背景显示即时变化
* **编辑颜色** → 自定义渐变立即应用**无需“应用”按钮**——所有更改均实时生效且交互式！

{% hint style="success" %}
**实时反馈**：即时可视化反馈让您能够快速尝试不同设置，直至找到满足分析需求的最佳可视化效果。
{% endhint %}

### 迭代优化工作流

**典型的 LUT 优化工作流：**

1.**选择索引图层**（例如 RAW（反射率））
2. **应用指数** - 选择相机滤镜和指数公式，将彩色圆圈拖动至指数公式中的适当位置
3. **应用 LUT 渐变** - 从 Red-Yellow-Green 预设开始
4. **检查像素值** - 移动光标，记录数值范围
5. **调整最小/最大值** - 缩小范围以聚焦植被（例如 0.2 至 0.9）
6. **选择裁剪** - 尝试使用“原始背景”以保留环境上下文
7. **优化色彩** - 如有需要，可自定义渐变以突出特定细节
8. **确定设置** - 记录设置并复制到“项目设置”中以便导出处理

### 像素值检查

了解实际像素值对于设置有效的 LUT 范围至关重要：

**如何检查值：**

1. 当图像的“索引”或“索引”和“LUT”**复选框均被勾选**时，会显示像素值。
2. **将光标移至**图像的不同区域
3. **观察**悬停时图例中显示的像素值
4. 放大以查看带有浮动值的单个像素
5. **记录**不同特征的数值范围：
   * **健康植被**：例如，NDVI 0.55-0.85
   * **受胁植被**：例如，NDVI 0.30-0.50
   * **裸露土壤**： 例如：NDVI 0.05-0.25
   * **水体**（若存在）：例如：NDVI -0.05 至 0.10**利用像素值设定 LUT 范围：**检查像素值后，据此调整 LUT 的最小/最大值：**示例场景：*** **观察结果**：土壤值 = 0.05-0.25，受胁迫 = 0.25-0.50，健康 = 0.50-0.85
* **目标**：仅可视化植物健康状况（排除土壤）
* **LUT 设置**：最小值 = `0.25`，最大值 = `0.85`
* **剪切**：选择“原始背景”以自然色显示土壤
* **结果**：颜色渐变仅应用于植被，土壤显示为原始图像

{% hint style="info" %}
**动态范围**：不同作物、季节和生长阶段的数值范围各不相同。在设置 LUT 范围前，请务必检查您特定数据集中的像素值。
{% endhint %}

## 自定义指数 (Chloros+)

### 创建自定义指数公式

{% hint style="info" %}
**创建位置**：可在处理前的**项目设置**中配置自定义指数，也可在图像查看器沙盒侧边栏中配置。
{% endhint %}

**创建自定义指数的方法：**

1.**打开项目设置**（处理前）或图像查看器沙盒侧边栏
2. 导航至 **索引公式下拉菜单**

3. 查找**“自定义”** 选项（必须使用 Chloros+ 许可证登录）
4. 使用波段变量**定义公式**：
   * 波段名称：`NIR`、`Red`、`Green`、`Blue`、`RedEdge` 等
   * 运算符：`+`、`-`、`*`、`/`、`^` (指数)
   * 函数：`sqrt()`、`abs()` 等（若支持）
   * 括号：`()` 用于运算顺序
5. **为索引命名**（例如“MyIndex”或“CustomNDVI”）
6. **保存配置**

**自定义公式示例：**

```

Modified NDVI with offset:
(NIR - Red) / (NIR + Red + 0.5)

Simple ratio:
NIR / Red

Complex multi-band:
(NIR - Red) / (NIR + Red - Blue)

Exponential index:
(NIR / Red) ^ 2
```

{% hint style="warning" %}
**公式验证**：请确保您的公式使用的波段在您的相机中可用。例如，RedEdge 仅适用于配备 RedEdge 滤光片的相机。
{% endhint %}
