# 图像图层

在Chloros图像查看器中，&quot;图像图层&quot;下拉菜单可让您快速切换同一图像的不同版本——从原始采集图像到处理后的反射率输出图及计算得到的指数图。

## 什么是图像图层？

在Chloros中，**图层**指单一源图像可生成的不同输出形式。图像处理过程中，Chloros会创建多个版本：

* **原始图像**（相机生成的JPG和RAW文件）
* **反射率校正**输出（若启用了反射率校正）
* **目标图像**（若图像包含校准目标）
* **索引图像**（若配置了索引，则生成NDVI、NDRE、GNDVI等）

图像查看器右上角的**图层选择器下拉菜单**可让您在不离开查看器的情况下即时切换这些版本。

***

## 可用图层类型

### JPG

* 相机原始JPG预览图像
* 所有图像均可使用
* 未处理的原始拍摄数据
* 加载显示速度最快

**适用场景：**

* 快速预览原始拍摄效果
* 检查画面构图与取景
* 处理前验证拍摄质量

### RAW（原始）

* 相机原始RAW传感器数据
* 完成去拜耳处理但未进行后期处理
* 比JPG更高位深度（通常为12位或14位传感器数据）

**适用场景：**

* 检查原始传感器数据质量
* 排查传感器问题或伪像
* 对比处理前后的效果

### RAW（校准目标）

* 仅在识别出含校准目标的图像时显示
* 呈现带目标标记的原始RAW图像
* 用于验证目标检测成功性

**适用场景：**

* 确认校准目标检测准确性
* 检查目标图像质量
* 排查校准问题

{% hint style=&quot;info&quot; %}
**目标图层**：仅在含校准目标的图像下拉菜单中显示。常规采集图像不提供此选项。
{% endhint %}

### RAW（反射率）

* 经校准的反射率输出图像
* 暗角校正 （若在处理中启用）
* 基于目标数据校正反射率（若启用）
* 多波段TIFF（含所有相机通道）
* 像素值代表百分比反射率（使用百分比模式时）
* 可通过[索引/LUT沙盒](index-lut-sandbox.md)进行处理

**适用场景：**

* 检查校准结果
* 验证校准质量
* 核查科学测量的像素值准确性
* 与原始图像对比校准效果

{% 提示 style=&quot;success&quot; %}
**推荐**：进行科学测量与分析时，请使用原始（反射率）图层检查像素值。
{% endhint %}

### 原始（NDVI 指数）...及类似图层

* 计算植被指数图像 （本例中为NDVI）
* 指数名称根据处理过程中配置的指数而变化
* 示例：RAW（NDVI指数）、RAW（NDRE指数）、RAW（GNDVI指数）等
* 单波段灰度图像，展示指数计算结果
* 项目设置中配置的每个指数对应一层

**可能的指数名称：**

* RAW (NDVI指数)
* RAW (NDRE指数)
* RAW (GNDVI指数)
* RAW (OSAVI指数)
* RAW (EVI指数)
* RAW (SAVI指数)
* 更多指数...（详见[多光谱指数公式](../project-settings/multispectral-index-formulas.md))

**适用场景：**

* 分析指数计算结果
* 核查指数数值范围
* 定位目标区域
* 在GIS或分析前验证指数图像

## 图层选择器使用指南

### 展开下拉菜单

1. 全屏打开图像（点击图像查看器任意缩略图）
2. 在查看器右上角定位**图层下拉菜单**
3. 下拉菜单显示当前选定图层（如&quot;JPG&quot;）
4. 点击下拉菜单查看所有可用图层

### 切换图层

1. 点击图层下拉菜单展开列表
2. 显示当前图像的所有可用图层
3. 点击任意图层名称即可切换至该版本
4. 图片将立即更新为所选图层

**快速切换技巧：**

* 下拉菜单会记住您的上次选择
* 浏览下一张图片时，系统将尝试显示相同图层类型
* 若下一张图片不存在该图层，则默认显示JPG

### 图层可用性

并非所有图层在每张图像中均可用：

**始终可用：**

* ✅ JPG（每张图像均有JPG预览）

**条件可用：**

* ⚠️ RAW（原始）- 仅当图像以RAW或RAW+JPG模式拍摄时可用
* ⚠️ RAW（目标）- 仅当图像包含检测到的校准目标时可用
* ⚠️ RAW（反射率）- 仅在启用反射率校准处理后可用
* ⚠️ RAW（\[索引]索引）- 仅在配置索引处理后可用

***

## 图层持久性

### 图片间导航

当您切换至其他图片（使用方向键或点击缩略图）：

**图层偏好设置将被保留：**

* 若当前查看&quot;RAW (反射率)&quot;，下一张图像将显示&quot;RAW (反射率)&quot;（若可用）
* 若当前查看&quot;RAW (NDVI 索引)&quot;，下一张图像将显示&quot;RAW (NDVI 索引)&quot; （若存在）
* 若不存在相同图层，则默认显示JPG图层

**示例工作流程：**

1. 打开图像1，切换至RAW (NDVI索引)图层
2. 按→键查看图像2
3. 图像2自动显示RAW (NDVI索引)图层
4. 继续浏览 - 所有图像均显示 NDVI 图层
5. 高效实现多图像索引结果比对

## 常用工作流程

### 工作流程 1：对比前/后效果

**目标**：比对原始图像与校准后图像

1. 在图像查看器中打开处理后的图像
2. 从下拉菜单选择**原始RAW**
3. 观察暗角现象与未校准数值
4. 切换至下拉菜单中的**反射率RAW**
5. 对比效果——暗角消除，数值完成校准

### 工作流程2：指数审查

**目标**：快速审查数据集中的NDVI结果

1. 打开首张处理图像
2. 从下拉菜单选择**原始数据（XPROTX索引图）**
3. 使用→方向键切换至下一张图像
4. XPROTX图层自动保持显示
5. 循环处理所有图像，核查XPROTX图案
6. 切换至**原始数据层（XPROTX索引）**进行对比

### 工作流3：目标验证

**目标**：验证所有目标图像是否正确检测

1. 导航至目标图像
2. 从下拉菜单选择**原始数据层（目标）**
3. 确认校准目标清晰可见且被检测
4. 导航至下一目标图像
5. 对所有目标重复验证流程

### 工作流程 4：像素值检查

**目标**：核查反射率值的科学准确性

1. 打开处理后的图像
2. 选择**RAW (反射率)**图层
3. 启用**像素百分比**模式（右上角工具栏按钮）
4. 将光标移至植被区域
5. 验证像素值是否在预期范围内（NIR为30-70%，Red为5-15%）
6. 检查土壤与水域区域是否符合相应数值

## 按图层解析像素值

不同图层显示的像素值范围各异：

### JPG图层

* **范围**：0-255（8位）
* **含义**：显示值，经伽马校正
* **用途**：仅供目视检查，不可用于科学测量

### RAW（原始）图层

* **范围**：0-65535（16位）
* **含义**：原始传感器数字值
* **用途**：检测传感器性能（未校准）

### RAW（反射率）

* **范围**：0-65,535（16位TIFF）或0.0-1.0（32位百分比）
* **含义**：校准后的百分比反射率
* **用途**：科学测量与分析

**16位TIFF格式：**除以65,535可得百分比反射率
**32位百分比格式：**数值直接代表百分比（0.5=50%反射率）

### RAW（指数图像）

* **范围**：随指数变化（归一化指数通常为-1.0至+1.0）
* **含义**：指数计算结果
* **示例**：
  * NDVI：-1至+1（植被指数通常为0.4至0.9）
  * NDRE：-1至+1（应力检测）
  * EVI：0至1（增强植被）

## 技巧与最佳实践

### 高效图层切换

* **键盘快捷键认知**：虽无图层专属快捷键，但方向键（←/→）可跨图层操作
* **统一工作流程**：选定单一图层（如NDVI）审阅完整数据集后再切换
* **快速比对**：在原始图与反射率图间切换以验证处理质量

### 性能考量

* **JPG层加载最快**：适用于快速浏览大量图像
* **RAW层加载较慢**：具备更高分辨率与位深度
* **索引层**：加载速度与反射率层相当
* **首次加载最慢**：同层后续查看将缓存加速

### 质量验证

* **务必检查原始RAW层**：确认源数据质量后再信任处理结果
* **对比图层**：通过图层切换验证处理流程正确性
* **检查索引范围**：使用索引层的像素百分比模式验证数值合理性

## 故障排除

### 图层不可用

**问题**：预期图层未出现在下拉菜单中

**可能原因：**

* 图像未处理（仅JPG和原始RAW可用）
* 处理过程中禁用了反射率校准
* 项目设置中未配置特定指数
* 图像为目标专用图像（未生成目标指数）

**解决方案：**

1. 确认图像已处理（检查输出文件夹中的处理文件）
2. 核查项目设置中是否配置了索引层
3. 重新处理时启用所需索引层

### 显示错误图层

**问题**：图像以非预期图层打开

**原因**：前次图像的图层偏好被继承，但当前图像不存在该图层

**解决方案：**Chloros在首选图层不可用时会自动回退至JPG格式——此为正常行为

### 校准目标不可见

**问题：**RAW（目标）图层未显示目标检测结果

**可能原因：**

* 处理过程中未检测到目标
* 图像实际不包含校准目标
* 目标检测设置过于严格

**解决方案：**

1. 在调试日志中检查&quot;目标已找到&quot;提示
2. 确认图像实际包含可见校准目标
3. 在项目设置中调整目标检测参数
4. 参阅[选择目标图像](../processing-images-gui/choosing-target-images.md)

## 相关功能

### 图像查看器工具

查看任意图层时可使用：

* **缩放控件**：放大以检查细节
* **平移**：点击拖动可在放大图像中移动
* **像素值检测**：查看光标位置的数值
* **导航箭头**：在保持图层状态下切换图像
* **像素百分比模式**：在数字值与百分比显示间切换

完整图像查看器文档请参阅[全屏打开图像](opening-an-image-full-screen.md)。

### 索引/LUT沙盒

用于交互式索引测试与可视化：

* **实时索引计算**：测试不同索引公式
* **LUT色彩映射**：为灰度索引应用色彩渐变
* **导出可视化效果**：保存彩色索引图像

详情参见[索引/LUT沙盒](index-lut-sandbox.md)。
