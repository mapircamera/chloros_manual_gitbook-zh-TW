# 地图标记

“地图”选项卡会根据图像的 GPS 坐标，在交互式 2D 地图上显示您的图像。这为您提供了拍摄 session 的地理概览，并有助于您直观地了解空间覆盖范围。在首次导入图像时，此功能也有助于快速移除无需处理的图像。

<figure><img src="../.gitbook/assets/chloros_map_markers.gif" alt=""><figcaption></figcaption></figure>

## 访问“地图”选项卡

1. 在 Chloros 中打开或创建一个项目
2. 导入包含 GPS 元数据的图像
3. 点击左侧边栏中的 **地图** <img src="../.gitbook/assets/image (3).png" alt="" data-size="line"> 标签
4. 地图将在每张图像的 GPS 位置显示标记

{% hint style="info" %}
**GPS 要求**：仅 EXIF 元数据中嵌有 GPS 坐标的图像才会显示在地图上。请确保拍摄时相机的 GPS 功能已开启。
{% endhint %}

## 图像标记

每张带有 GPS 数据的图像在地图上均由一个标记表示：

### 标记显示

* 标记显示了每张图片拍摄时的精确GPS坐标
* 缩小地图时，相邻的标记可能会聚集成组
* 放大地图可查看单张图片的具体位置

{% hint style="success" %}
超级缩放：当您达到地图图块提供商设定的最大缩放级别后，继续放大时图块会自动放大，从而让您能够看到紧密相邻的标记。
{% endhint %}

### 悬停预览

* **将鼠标悬停**在任何标记上，即可查看该图像的缩略图预览
* 这允许您在不离开地图视图的情况下快速进行视觉识别
* 对于在大型拍摄会话中定位特定图像非常有用

***

## 地图图块提供商

{% hint style="success" %}
**自动选择**：Chloros 会自动选择能为当前地图位置提供最佳缩放级别的图块服务。如需切换，您可手动选择不同提供商。
{% endhint %}

“地图”选项卡支持两种用于背景地图影像的图块提供商：

### Google 地图

* 来自 Google 的标准卫星和地图影像
* 最适合全球范围的通用覆盖

### ESRI

* 来自 ESRI ArcGIS 的卫星和航空影像
* 在某些地区通常提供更高分辨率的影像

## 地图图块类型

您可以选择地图图层类型（从左到右）：

 <img src="../.gitbook/assets/image (23).png" alt="" data-size="original">### 地形

### 地图

显示标准（低带宽）且包含细节（道路等）的地图图块

显示详细（高带宽）的卫星地图图块

### 混合

显示带有附加细节（道路等）的卫星地图图块

## 应用场景

### 飞行路径可视化

* 查看无人机拍摄任务的覆盖区域
* 识别图像覆盖中的空白区域
* 验证飞行路径执行情况

### 地面勘测审查

* 查看地面拍摄数据的空间分布
* 定位校准目标图像相对于勘测区域的位置
* 规划额外的拍摄点

### 质量控制

* 快速识别在意外位置拍摄的图像
* 验证数据集中的 GPS 精度
* 将图像位置与实地记录进行交叉核对

***

## 故障排除

### 未显示标记

**可能原因：**

* 图像不包含 GPS 元数据
* 拍摄时相机上的 GPS 功能被禁用
* EXIF 数据被外部软件删除

**解决方案**：确认相机已启用 GPS 功能，并重新导入原始文件

### 标记位置错误

**可能原因：**

* 相机 GPS 卫星定位效果不佳
* 拍摄过程中 GPS 发生漂移

**解决方案**：这通常是拍摄时的问题；对于高精度应用，建议使用 PPK/RTK GPS
