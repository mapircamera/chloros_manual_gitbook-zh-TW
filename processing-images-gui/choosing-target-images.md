# 选择目标图像

标记包含校准目标的图像至关重要，此步骤能显著加速Chloros处理流程。通过预先筛选目标图像，可避免Chloros对数据集中的每张图像进行校准目标扫描。

## 为何标记目标图像？

### 提升处理速度

若未标记目标图像，Chloros必须执行以下操作：

* 扫描项目中的每张图像
* 对每张图像运行目标检测算法
* 无谓地检查数百甚至数千张图像

**结果**：处理时间将显著延长，尤其针对大型数据集。

### 标记目标图像时

当您勾选特定图像的&quot;目标&quot;列时：

* Chloros仅扫描已勾选图像中的目标
* 目标检测速度大幅提升
* 整体处理时间显著缩短

{%提示 style=&quot;success&quot; %}
**速度提升**：在500张图像的数据集中标记2-3张目标图像，可将目标检测时间从30多分钟缩短至1分钟以内。
{% endhint %}

***

## 如何标记目标图像

### 步骤1：识别目标图像

在文件浏览器中查看导入的图像，确定哪些图像包含校准目标。

**常见场景：**

* **预捕获目标**：在开始会话前拍摄
* **后捕获目标**：在完成会话后拍摄
* **现场目标**：放置在捕获区域内的目标
* **多目标**：每会话2-3张目标图像（推荐）

### 步骤二：核对目标列

针对每张含校准目标的图像：

1. 在文件浏览器表格中定位该图像
2. 找到**目标**列（最右侧列）
3. 勾选该图像对应目标列的复选框
4. 对所有含目标的图像重复此操作

### 步骤三：验证选择结果

处理前请双重确认：

* [ ] 所有含校准目标的图像均已勾选
* [ ] 未误选非目标图像
* [ ] 目标在所选图像中清晰可见

## 目标图像最佳实践

### 目标采集指南

**时机：**

* 拍摄目标图像应紧接在采集会话开始前及整个采集过程中进行
* 确保与DAQ光传感器处于相同光照条件下
* 尽可能高频采集目标图像以获得最佳效果。否则系统将随时间推移使用光传感器数据进行校准调整

**相机定位：**

* 将相机置于目标上方，使目标居中并占据图像中心区域40-60%
* 保持相机与目标表面平行/垂直

**光照条件：**

* 环境光照需与DAQ光传感器一致
* 避免目标表面产生阴影
* 勿用身体、车辆或植被遮挡光源
* 多云天气可获得最稳定结果

**目标状态：**

* 保持目标面板清洁干燥
* 四块面板均需清晰可见且无遮挡
* 目标面尽可能与光源垂直/垂直下方

### 目标图像数量要求？

**最低要求：**每次采集至少1张目标图像。**推荐：**每次采集3-5张目标图像。

**最佳实践流程：**

* 光传感器开始记录后立即采集3-5张图像
* 每次采集间隔旋转相机以获得最佳效果
* 可选：若光照条件持续变化，可在采集中途定期补充拍摄

***

## 多相机协同工作

### 双摄像头配置

若同时使用两台MAPIR摄像头（例如Survey3W与RGN组合）：

1. 使用**两台相机**同时捕捉目标图像
2. 两台相机使用**同一物理目标**
3. 在文件浏览器中为**两种相机类型**的目标图像添加标记
4. Chloros将为每台相机的校准使用相应目标

### 相机型号列

**相机型号**列用于区分图像来源：

* Survey3W\_RGN
* Survey3N\_OCN
* Survey3W\_RGB
* 等

请通过此列核查项目中是否为每种相机类型标记了目标。

## 目标检测设置

### 调整检测灵敏度

若Chloros无法正确检测目标，请在[项目设置](adjusting-project-settings.md)中调整以下参数：

**最小校准采样区域：**

* **默认值**：25像素
* 若小尺寸工件出现误检，请**增大**此值
* 若目标未被检测，请**减小**此值

**最小目标聚类：**

* **默认值**：60
* **增大**：当目标被分割为多个检测结果时
* **减小**：当存在色差的目标未被完整检测时

***

## 常见目标图像问题

### 问题：未检测到目标

**可能原因：**

* 目标图像未在文件浏览器中标记
* 目标在画面中过小（&lt;图像面积的30%）
* 光线条件差（阴影/眩光）
* 目标检测设置过严

**解决方案：**

1. 确认目标列已勾选正确图像
2. 预览目标图像质量
3. 质量不佳时重新采集目标
4. 必要时调整目标检测设置

### 问题：虚假目标检测

**可能原因：**

* 白色建筑、车辆或地面覆盖物被误判为目标
* 植被中的高亮区域
* 检测灵敏度过低

**解决方案：**

1. 仅标记实际目标图像以缩小检测范围
2. 增加最小校准采样面积
3. 提高最小目标聚类值
4. 确保目标图像仅显示目标（背景干扰最小）

***

## 验证检查清单

开始处理前请核对目标图像选择：

* [ ] 每轮次至少标记1张目标图像
* [ ] 所有目标图像均勾选目标列复选框
* [ ] 目标图像采集时间与测绘同步
* [ ] 点击预览时目标清晰可见
* [ ] 每张目标图像均显示全部4个校准面板
* [ ] 目标区域无阴影或遮挡
* [ ] 双摄像头模式：两种摄像头类型均标记目标

***

## 无目标处理

### 禁用校准目标的处理方式

虽不推荐用于科研工作，仍可执行无目标处理：

1. 保持所有目标列复选框未勾选
2. 在项目设置中**禁用**&quot;反射率校准&quot;
3. 仍将应用晕影校正
4. 输出结果不进行绝对反射率校准

{% 提示 style=&quot;warning&quot; %}
**不推荐**：未进行反射率校准时，像素值仅代表相对亮度，而非科学反射率测量值。请使用校准目标以获得准确、可重复的结果。
{% endhint %}
