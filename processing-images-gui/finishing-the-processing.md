## 处理完成指示

当处理成功完成时，您将看到以下指示：

* ✅ **进度条**：达到100%完成度
* ✅ **调试日志**：显示&quot;处理完成&quot;消息

## 常见输出问题及解决方案

### 问题：输出文件缺失

**可能原因：**

* 文件未满足处理标准
* 仅目标图像（被排除在导出之外）
* 导出过程中磁盘空间不足
* 处理过程中文件损坏

**解决方案：**

1. 检查调试日志中的跳过/错误信息
2. 确认磁盘空间充足
3. 核对文件数量：应与（原始文件数 - 目标文件数）×（索引值+1）相符
4. 重新导入并处理缺失文件

### 问题：暗角或亮边（晕影仍可见）

**可能原因：**

* 未启用暗角校正功能
* 相机/镜头未收录于Chloros配置文件数据库
* 暗角程度超出校正能力范围

**解决方案：**

1. 确认项目设置中已启用暗角校正
2. 检查相机型号是否正确识别
3. 若暗角问题持续存在，请联系MAPIR技术支持

### 问题：颜色或数值异常

**可能原因：**

* 未检测到校准目标
* 选择错误的校准目标型号
* 反射率校准功能未启用
* 目标图像质量不佳

**解决方案：**

1. 确认反射率校准功能已启用
2. 检查调试日志中的&quot;目标已找到&quot;提示
3. 复核目标图像质量
4. 标记正确目标后重新处理

### 问题：NDVI数值异常

**预期NDVI数值范围：**

* **水体、岩石、土壤**：-0.1至0.2
* **稀疏/不健康植被**：0.2至0.4
* **中等植被**：0.4至0.6
* **健康密集植被**：0.6至0.9

**若数值超出上述范围：**

1. 确认反射率校准已应用
2. 确认光传感器日志已包含
3. 检查校准目标是否被检测到
4. 确保检测到正确相机型号
5. 复核目标图像采集时间与环境条件

## 处理后图像的应用

### 用于摄影测量/正射镶嵌图生成

**推荐工作流程：**

1. **导入校准反射率图像**至摄影测量软件：
   * Pix4Dmapper
   * Agisoft Metashape
   * DroneDeploy
   * WebODM
2. **保留EXIF元数据**：确保GPS数据完整用于地理标记
3. **校准工作流**：使用反射率图像确保科学精度
4. **处理索引图拼接**： 从单张索引图像生成NDVI正射镶嵌图
5. **导出地理参考GeoTIFF**：用于GIS应用

### GIS分析流程

**推荐工作流程：**

1. **加载至QGIS、ArcGIS或同类软件**
2. **使用16位TIFF**反射率图像进行多波段分析
3. **使用指数图像**（NDVI, NDRE）作为现成植被图层
4. **栅格计算器**：组合波段进行定制分析
5. **导出功能**：生成分类图、变化检测图、植被健康图

### 直接分析/报告应用

**推荐工作流：**

1. **采用带LUT色彩的指数图像**制作可视化报告
2. **提取统计数据**：按田块/样地计算NDVI均值
3. **时间序列**：跨多次观测对比指数变化
4. **生成报告**：包含地图、统计数据及可视化图表

## 后续处理流程

### 重用项目设置

若未来处理类似数据集：

1. **保存项目模板**（若尚未操作）
2. 使用保存的模板**创建新项目**
3. **导入新图像**
4. 采用相同设置**进行处理**以保持一致性

### 多会话批量处理

针对多个会话/数据集：

**方案1：GUI - 多项目处理**

* 为每个会话创建独立项目
* 使用统一的模板设置
* 逐个处理

**方案二：Chloros CLI（仅限Chloros+版本）**

* 自动化批量处理
* 通过脚本处理多个文件夹
* 参见[CLI文档](../CLI.md)

**方案3：Python SDK（仅限Chloros+）**

* 程序化控制
* 与分析管道集成
* 参见[API 文档](../api-python-sdk.md)

## 后处理故障排除

### 采用不同设置重新处理

若结果不理想：

1. 保留原始图像（切勿删除）
2. 在Chloros中打开相同项目
3. 调整项目设置面板中的参数
4. 重新处理——输出将覆盖先前结果

### 处理图像子集

仅需重新处理特定图像时：

1. 创建新项目
2. 仅导入需重新处理的图像
3. 使用相同设置模板
4. 处理较小数据集
