# 输出图像格式

### TIFF (16位)

16位（像素数值 0 - 65535）TIFF (.tif) 格式输出。

### TIFF (32 位，百分比 %)

32 位（像素浮点数 0.0 - 1.0）TIFF (.tif) 格式输出。像素值表示反射率百分比，其中 0.0 = 0%，1.0 = 100%。

### **JPG (8 位)**

8 位（像素数字 0 - 255）JPG (.jpg) 格式输出。

### **PNG (8位)**

8位（像素数字 0 - 255）PNG (.png) 格式输出。无损压缩，适用于网页浏览和可视化。
